### 문제 요약

- n이 주어졌을 때 n번째 피보나치 수를 1234567로 나눈 나머지를 반환

## 핵심 포인트

n의 최댓값은 100,000으로, 100,000번째 피보나치 수를 구해야 하는 경우가 존재한다. 하지만, 100,000번째 피보나치 수는 매우 큰 값으로 int는 물론, long의 범위도 벗어나는 큰 수이다. 따라서, 단순히 값을 구해 모듈러 연산으로 나머지를 구하더라도 이상한 답이 나오게 된다.

이를 해결하기 위해선 아래와 같은 법칙을 활용해야 한다.

```java
(A + B) % C = (A % C + B % C) % C
(A - B) % C = (A % C - B % C) % C
(A * B) % C = (A % C * B % C) % C
```

이 법칙을 활용하여 중간 연산 결과가 int 범위를 넘지 않도록 제한을 두어야 한다.

### 전체 코드

```java
class Solution {
    public long solution(int n) {
        long[] dp = new long [n+1];
        dp[0] = 0;
        dp[1] = 1;
        for (int i = 2; i <= n; i++)
            dp[i] = (dp[i-1] + dp[i-2]) % 1234567;

        return dp[n];
    }
}
```
