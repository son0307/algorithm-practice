### 문제 요약

- 기지국이 설치된 위치와 기지국의 전파 거리가 주어졌을 때, 아파트 모든 동에 전파가 도달할 수 있도록 증설해야 하는 기지국의 최소 대수를 계산하는 문제
    - 각 기지국의 전파는 설치 위치로부터 좌우로 W만큼 도달할 수 있다.

## 핵심 포인트

설치해야할 기지국의 수를 최소화하려면 전파의 도달 가능 범위를 최대한 겹치지 않도록 설치해야 한다. 예를 들어, 기지국의 전파 범위 W가 2이고 총 7개의 아파트를 커버해야할 때, 필요한 기지국의 숫자는 다음과 같이 계산할 수 있다.

- 기지국 1개 → 5개 아파트 커버, 추가 기지국 1개 → 나머지 2개 아파트 커버

이렇게 총 2개의 기지국을 사용해야한다. 이를 일반화하여 생각해보도록 하자. 기지국의 전파 범위가 W일 때, 최대로 커버할 수 있는 아파트는  2 * W + 1 개이다. 아파트의 개수를 N개라 한다면 필요한 기지국의 수는 다음과 같다.

- N / (2 * W + 1) + (나머지 존재 시 1, 없으면 0)

이를 바탕으로 문제를 접근해보자.

문제의 목적은 커버되고 있지 않은 부분을 커버하는 것이므로 이미 설치된 기지국을 통해 커버되고 있는 범위 사이의 거리를 구하고, 위 식을 적용하여 필요한 기지국들의 수를 구하면 된다. 커버해야할 범위를 구하는 방식은 다음과 같다.

- 아파트 첫 동부터 순차적으로 탐색 (current를 1로 설정한다.)
- 각 아파트의 상태는 커버되고 있는 경우와 커버되고 있지 않는 경우가 있다.
    - 커버되고 있는 상태 → 커버 범위를 벗어나기

      ![image.png](attachment:0f645c0a-a481-485e-b6ce-83df30450d7d:image.png)

        - 위 예시의 경우, 2번에 기지국이 설치되어 1~3의 거리를 커버하고 있다. 그러므로, 2(기지국의 위치) + 1(커버 범위) + 1 을 통해 범위를 벗어나 커버되지 않는 범위의 시작점으로 이동시킨다.
    - 커버되고 있지 않는 상태 → 다음 커버 범위까지의 거리를 구하고 그 범위를 커버하기 위한 최소한의 기지국 개수를 계산

      ![image.png](attachment:77f5269f-31e1-49fc-bab5-96c0a43a375c:image.png)

        - 위 예시의 경우, 6번에 기지국이 설치되어 5~7의 거리를 커버하고 있다. 현재 current는 그 범위에 포함되어 있지 않으므로 1~4의 거리를 커버할 수 있도록 필요한 기지국의 최수 개수를 계산하면 된다. 이후 해당 범위 다음으로 이동하여 탐색을 계속 수행한다.
- 위 과정을 모든 아파트에 대해 계산할 때까지 반복한다.

![image.png](attachment:878b974b-37b1-4bd3-a47f-e45f81578205:image.png)

이를 예시로 전체적인 처리 과정을 살펴보자.

1. current를 1로 설정한다.
2. 처음 만나게 될 기지국이 커버하고 있는 범위를 확인한다. 8번 위치에 있는 기지국의 커버 범위가 7~9이므로 1번은 포함되지 않는다.
3. 1~6의 범위를 커버할 때 필요한 기지국의 개수를 계산한다. 이후, current를 7로 설정한다.
    - 6 / (2 * 1 + 1) = 2 → 정답에 2 추가
4. current가 7이면 커버 범위에 속하므로 벗어나도록 current를 10으로 설정한다.
5. 마지막에 커버되지 않은 부분이 존재하므로 이 부분도 커버할 수 있도록 필요한 기지국의 개수를 계산한다.
    - 1 / (2 * 1 + 1) = 0, 나머지가 존재하므로 +1 → 정답에 1 추가

필요한 기지국의 최소 개수는 3개이다.

### 전체 코드

```groovy
class Solution {
    public int solution(int n, int[] stations, int w) {
        int answer = 0;
        int current = 1;
        int range = 2 * w + 1;

        for (int station : stations) {
            int start = station - w;
            if (current < start) {
                int dist = start - current;
                answer += dist / range;
                if (dist % range != 0)
                    answer++;
            }

            current = station + w + 1;
        }

        if (current <= n) {
            int dist = n - current + 1;
            answer += dist / range;
            if (dist % range != 0)
                answer++;
        }

        return answer;
    }
}
```
